Features

Everything in one stack

Five Docker containers, one docker-compose.yml, zero external dependencies.

🧠

Private LLM

Ollama-powered local inference. Runs on GPU or CPU. Fully offline — your data never leaves the building.

💬

Web Chat UI

Open WebUI with conversations, file uploads, model switching. Accessible from any browser.

📄

RAG — Chat Over Documents

Drop PDFs, docs, or text files. AI answers based on your documents, citing the source. No external vector DB needed.

🤖

Telegram Bot

AI assistant for your team inside Telegram. Gated by user ID allowlist — only your people get access.

🔐

VPN Included

Tailscale sidecar for secure remote access (requires free Tailscale account).
