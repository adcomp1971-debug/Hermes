для вашего офиса

Все компоненты уже существуют по отдельности — Ollama, Open WebUI, Tailscale. Сложность в том, чтобы собрать их вместе: соединить контейнеры, настроить Telegram-бота с белым списком, сделать RAG который не галлюцинирует, добавить VPN без открытых портов. Обычно это день танцев с бубном. Hermes Box — уже собранный стек, который работает за 5 минут.

Никаких облаков, никакой платы за пользователя.

Возможности

Всё в одном стеке

Пять Docker-контейнеров, один docker-compose.yml, ноль внешних зависимостей.

🧠

Приватный LLM

Локальный инференс через Ollama. На GPU или CPU. Полностью офлайн — данные не покидают офис.

💬

Веб-чат

Open WebUI с историей, файлами, переключением моделей. Доступно из любого браузера.

📄

RAG — чат по документам

Кидайте PDF, DOC, TXT. AI отвечает строго по вашим документам с указанием источника. Если ответа нет — честно отказывается, вместо того чтобы выдумывать. Без внешних векторных БД — просто numpy на диске.

🤖

Telegram-бот

AI-ассистент для команды в Telegram. Доступ только по белому списку ID.

🔐

VPN в комплекте

Tailscale для удалённого доступа (нужен бесплатный аккаунт Tailscale).

Установка на ваше железо

3 команды до работающего стека, затем кидайте документы.

git clone https://github.com/adcomp1971-debug/Hermes && cd Hermes && ./setup.sh && docker exec hermes-ollama ollama pull qwen2.5:7b

Первая загрузка модели ~4.7 GB — 5–15 мин на CPU, быстрее с GPU.
