Self-hosted AI for your office

All the pieces already exist — Ollama, Open WebUI, Tailscale. The hard part is assembly: wiring containers, securing the Telegram bot, setting up RAG that doesn't hallucinate, adding VPN without opening ports. Hermes Box is a pre-assembled stack that works in 5 minutes. No cloud, no per-seat fees.

Features

Everything in one stack

🧠 Private LLM

Ollama-powered local inference. Runs on GPU or CPU. Fully offline — your data never leaves the building.

💬 Web Chat UI

Open WebUI with conversations, file uploads, model switching. Accessible from any browser on your network.

📄 RAG — Answers From Your Documents

Drop PDFs, docs, or text files. AI searches them and answers with a visible citation. If the answer isn't there, it says so instead of making things up. No external vector DB needed — just numpy on disk.

🤖 Telegram Bot

AI assistant for your team inside Telegram. Gated by user ID allowlist — only your people get access.

🔐 VPN Included

Tailscale sidecar for secure remote access (requires free Tailscale account).

⚡ 3 Commands + Docs

git clone + ./setup.sh + ollama pull. Then drop your documents and run POST /ingest.
